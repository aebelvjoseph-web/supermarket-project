# ARBHA Supermarket Website

Welcome to the official source code for the **ARBHA Supermarket** informational website. 

This project has been simplified into a **Clean Static Structure** (HTML, CSS, and JS) to ensure it is fast, secure, and easy to maintain.

## 📁 Project Structure

- `index.html`: The Home page with featured categories and the main welcome message.
- `products.html`: The full Product Catalog including Vegetables, Grocery, Books & Stationery, and more.
- `offers.html`: A dedicated page for Special Offers and discounts.
- `about.html`: Information about the history and mission of ARBHA Supermarket.
- `contact.html`: Contact details and order instructions (+91 97478 30154).
- `location.html`: Store location details (Opposite CRPF Camp, Aravanchal).
- `style.css`: The "Design Brain" - controls all the colors, fonts, and the premium look.
- `script.js`: Controls the smooth animations and page interactions.

## 🚀 How to Edit

1. **Changing Text**: Open any `.html` file in a text editor (like Notepad or VS Code) and simply type your new text between the tags.
2. **Changing Prices/Categories**: You can update the category headings in `products.html`.
3. **Changing Colors**: All design tokens are at the top of `style.css` under `:root`.

## 🌐 Hosting
